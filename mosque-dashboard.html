SMART WiFi MOSQUE DASHBOARD

Mosque Dashboard is a fully automated, Smart Dashboard for Mosques.
It displays Jamaat Salah times, daily Sehri & Iftaar times for Fasting, Ishraq and Chasht nafil namaz time, Gregorian and Hijri dates, Current temperature at the Masjid, Current day's Moon phase and Current time; down precise to Seconds.

How it Works?

Completely Local System

Entire Mosque Dashboard system including Server and other components are built into Local Controller. Controller is full fledged Independent system.

Temperature Sensor

Mosque Dashboard comes with its own Temperature controller which regularly senses current temperature in Masjid and displays it on Dashboard.

Choice of Screen Size

Dashboard System uses a Full HD/HD Ready TV with HDMI port to display the Content.
You have an option to choose from various different  sizes available.

Data Update through Android App

Mosque Dashboard System uses an Android App installed on your Smartphone to update or change data like Namaz times, Hijri calendar adustment etc.

Jamaat time Alerts

When it is time for Jamaat to stand up, a System plays a Bismillah audio to alert Namazis.
In addition, the Jamaat time displayed on Dashboard corresponding to current Jamaat flashes on the screen for 15 seconds to indicate time for Jamaat.

Hijri Date & Moon phase

System displays current Hijri Date on Dashboard.
It also displays current phase of the Moon in full 3D real life images of moon.

Current Tempreature

Dashboard system displays real time current tempreature in the Masjid at all times.
The tempreature display is Color coded, tempreature is displayed in different colors based on how hot or cold it is.

Configuration

Display

Mosque Dashboard uses a Full HD/HD Ready TV as Display. We will provide you with the size of HD/HD Ready TV that you choose while placing order.
For details of cost involved, click here.

Controller

Mosque Dashboard uses a Microcontroller based System, which has built in Servers and other Software installed to accurately calculate Namaz time data, Hijri Date and automatically fetch current temperature in Masjid and display these on the Dashboard.
It also plays Bismillah audio when it is time for a Namaz.

Android App for Data update

Mosque Dashboard has an Android app for changing Namaz times and other data on the Display.
This App is available at Play Store and can be installed on any Android Smartphone with Android 4.0 or above.
It is also possible to update Dashboard data without installing the App, using any Internet Browser like Chrome.
